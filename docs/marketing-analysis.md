| **月額課金** | $5-15 | $6.99 | $4.99-19.99 |
| **転換率** | 3-5% | 15% | 8-10% |
| **解約率** | 5-10% | 2-3% | 5% |

### Fujisan.AI LTV計算

```
【Proプラン ($9.99/月) の場合】

平均継続期間: 10ヶ月
月額: $9.99
LTV = $9.99 × 10 = $99.90

【年額プラン ($89.88/年) の場合】
継続率: 70%
1年目: $89.88
2年目: $89.88 × 0.7 = $62.92
LTV = $89.88 + $62.92 = $152.80

【目標CAC】
LTV:CAC = 7:1 とすると
CAC = $99.90 / 7 = $14.27
```
