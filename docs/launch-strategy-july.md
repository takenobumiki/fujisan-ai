# JLPT本気組を獲得する戦略

## 課題の整理

### タイムライン
```
2025年
1月 ────── 7月 ────── 12月
 │          │          │
 │          │          └─ JLPT 12月試験
 │          └─ JLPT 7月試験 → 合格発表8-9月
 └─ 今ここ
 
【問題】
・7月試験の合格体験記は8-9月まで出ない
・それまで「実績」を証明できない
・有料で売るには信頼性が足りない
```

---

## 戦略オプション比較

### Option A: 7月まで完全無料

| 項目 | 内容 |
|------|------|
| 期間 | 2025年1月〜7月（6ヶ月） |
| 対象 | 全機能無料開放 |
| 目的 | ユーザー獲得、合格者輩出 |

**メリット**:
- ✅ 参入障壁ゼロ → 最大のユーザー獲得
- ✅ 7月試験の合格者を確実に作れる
- ✅ 合格体験記が8月から使える
- ✅ 口コミが広がりやすい

**デメリット**:
- ❌ 6ヶ月間収益ゼロ
- ❌ 「無料だから使う」層が多い（課金転換難）
- ❌ 無料→有料の切り替えで離脱リスク

**収益シミュレーション**:
```
1-6月: 収益 $0
7月以降: 無料ユーザー5,000人 × 転換率5% = 250人
         250人 × $10/月 = $2,500/月〜

年間予測: 約$15,000〜$30,000
```

---

### Option B: 7月まで「フリーミアム強化」

| 項目 | 内容 |
|------|------|
| 無料枠 | 1日10問まで無料（現状より拡大） |
| 有料 | 無制限 + AI機能 |
| 特別 | 7月試験合格で全額返金 or 1年無料 |

**メリット**:
- ✅ 収益を維持しながらユーザー獲得
- ✅ 本気組は有料でも払う
- ✅ 「合格保証」が強力なUSP

**デメリット**:
- ❌ 無料枠だけで満足する層も
- ❌ 返金リスク

**収益シミュレーション**:
```
1-6月: 有料ユーザー500人 × $10 = $5,000/月
       累計 $30,000
7月以降: 合格者に返金（-$10,000想定）
         新規増加で回復

年間予測: 約$50,000〜$80,000
```

---

### Option C: 7月試験組限定「合格保証プログラム」

| 項目 | 内容 |
|------|------|
| 対象 | 7月JLPT受験予定者 |
| 価格 | $49.99（6ヶ月パック） |
| 保証 | **不合格なら全額返金** |
| 特典 | 合格したら次レベル6ヶ月無料 |

**メリット**:
- ✅ 「本気組」だけがターゲット
- ✅ 返金保証で参入障壁を下げる
- ✅ 収益を確保しつつリスクは限定的
- ✅ 合格者は継続課金の可能性高い

**デメリット**:
- ❌ 返金対応の運用コスト
- ❌ 不合格者が多いと赤字

**収益シミュレーション**:
```
1-6月: 1,000人 × $49.99 = $50,000
7月: 合格率70%想定 → 300人返金 = -$15,000
     合格者700人 → 次レベル継続
     
純収益: $35,000
+ 継続課金: 700人 × $60/6ヶ月 = $42,000

年間予測: 約$77,000
```

---

### Option D:「本気組」向け無料 + 一般は有料

| 項目 | 内容 |
|------|------|
| 条件 | 7月JLPT受験票の写真提出で無料 |
| 期間 | 試験日まで全機能無料 |
| 義務 | 合格したら体験記を書く（任意だが推奨） |

**メリット**:
- ✅ 本当の「本気組」だけを獲得
- ✅ 受験票という証明で不正防止
- ✅ 合格体験記の確保
- ✅ 一般ユーザーからは収益

**デメリット**:
- ❌ 受験票は4-5月発送 → それまで適用不可
- ❌ 運用が複雑

## まとめ：7月までの戦略

### 結論

**「7月まで完全無料」ではなく「ハイブリッド型」を推奨**

| 施策 | 理由 |
|------|------|
| 先着1,000人無料 | 初期ユーザー確保、口コミ |
| 合格保証プラン | 本気組のコミットメント獲得 |
| 一般有料 | 収益確保 |

### 期待効果

| 指標 | 7月時点 | 12月時点 |
|------|---------|----------|
| 累計ユーザー | 2,000人 | 5,000人 |
| 有料ユーザー | 800人 | 2,500人 |
| 合格者数 | 500人+ | 1,500人+ |
| 累計収益 | $20,000 | $180,000 |
| 合格体験記 | 100件+ | 500件+ |

### 最重要KPI

> **7月試験で何人合格させられるか**

これが8月以降の全てを決める。

合格者が多ければ:
- 合格体験記が集まる
- 「合格できるアプリ」として認知
- 口コミで拡散
- 12月試験組が殺到

**だから、7月試験組を無料でも獲得し、徹底サポートする価値がある。**
