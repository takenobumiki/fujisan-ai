## なぜ少ないか（正直な理由）

### 1. 認知ゼロの壁
```
見たこともないアプリ
    ↓
「本当に大丈夫？」という不安
    ↓
無料トライアルでも躊躇
    ↓
課金はさらにハードル高い
```

### 2. Web/PWAの不利
| 項目 | ネイティブアプリ | Web/PWA |
|------|-----------------|---------|
| 発見 | App Store検索 | 検索/広告のみ |
| 信頼 | レビュー・DL数が見える | 見えない |
| インストール | ワンタップ | ブックマーク（面倒） |
| 通知 | 確実 | 許可必要 |

### 3. 14日トライアルの長さ
```
1月1日 トライアル開始
    ↓
1月14日 トライアル終了
    ↓
1月15日以降 課金判断
    ↓
実質、課金が発生するのは月後半
```

### 4. JLPT試験時期
- 7月試験まで6ヶ月
- 「まだ早い」と思う人も
- 12月試験は終わったばかり

## 各ケースでの判断

### 楽観ケース（12人）→ 🟢 継続

**意味**: 
- 実績ゼロでも買う人がいる
- プロダクト/価格は受け入れられている
- 広告を増やせばスケールの可能性

**2月アクション**:
- 広告予算$1,500-2,000に増額
- うまくいった広告を拡大
- そのまま定価継続

---

### 現実ケース（5-8人）→ 🟡 微妙

**意味**:
- 買う人はいるが、スケールは不透明
- 認知/信頼の壁が大きい
- 無料施策で加速した方が良いかも

**2月アクション**:
- 「7月合格応援キャンペーン」検討
- 先着300-500人無料など
- または合格保証プラン

---

### 悲観ケース（1-2人）→ 🔴 要検討

**意味**:
- 定価では厳しい
- プロダクト/価格/訴求のどれかに問題
- または市場自体が小さい

**2月アクション**:
- 広告一旦停止
- 無料ユーザーからフィードバック収集
- プロダクト改善に集中
- 3月以降に再挑戦

## この結果をどう捉えるか

### ポジティブな見方

| 結果 | 意味 |
|------|------|
| 5人でも課金 | 「買う人は存在する」証明 |
| トライアル50人 | 興味を持つ人はいる |
| 継続率がわかる | LTV予測の材料 |
| どの広告が効くか | 2月以降に活かせる |

### 1月の本当の目的

> **売上を作ることではなく、**
> **「売れる可能性があるか」を確認すること**

5-8人でも：
- 買う人が存在する → 市場はある
- 転換率10%以上 → プロダクトは悪くない
- 特定の国/層が反応 → ターゲットが絞れる

## 結論

### 予測

| 指標 | 1月末予測 |
|------|----------|
| **有料転換** | **5-8人** |
| **MRR** | **$50-80** |
| **判定** | **🟡 微妙だが継続可能** |

### 5人でも成功と言える条件

1. トライアル登録が50人以上ある
2. 転換率が10%以上ある
3. 特定のチャネル/国で効果が見える
4. ユーザーからポジティブな声がある

**これらが確認できれば、2月から無料施策で加速する価値がある。**
