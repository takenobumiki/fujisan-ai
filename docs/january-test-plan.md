# Fujisan.AI 1月定価テスト計画

**1ヶ月間、定価で販売し「売れるかどうか」を検証する**

| 指標 | 🔴 撤退検討 | 🟡 要改善 | 🟢 継続 | 🚀 加速 |

### 判断フロー

```

    │
    ├─ 有料30人以上 → 🚀 このまま加速
    │
    ├─ 有料16-29人 → 🟢 継続しつつ価格/機能調整
    │
    ├─ 有料6-15人  → 🟡 2月から無料キャンペーン検討

    └─ 有料0-5人   → 🔴 大幅な戦略見直し
                      ・価格引き下げ
                      ・無料期間延長
                      ・ターゲット変更
                      ・プロダクト改善
```
